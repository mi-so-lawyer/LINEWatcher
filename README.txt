LINEWatcher v8 - 実行ファイル版（PDF対応）

このバージョンでは、拡張子のないファイルを中身から自動判定して、
JPEG だけでなく PDF にも正しくリネーム＆保存できます。

【含まれるファイル】
- LINEWatcher_v8.exe ............. 実行ファイル本体（PDF対応）
- install_startup.bat ............ スタートアップ登録用
- remove_startup.bat ............. スタートアップ解除用
- create_shortcut.vbs ............ バッチ用ショートカット作成補助
- linewatcher_icon.ico ........... 実行ファイル用アイコン

【使い方】
1. 実行ファイルを起動（初回はフォルダ選択）
2. 「名前を付けて保存」で画像やPDFを保存
3. 自動でリネーム＆移動、通知で完了！

詳しい解説や使い方は GitHub / note を参照してください。
